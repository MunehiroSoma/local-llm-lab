# 観点: 外部API連携

> [`review`](../../../.claude/skills/review/SKILL.md) スキルの観点別サブエージェントが読む単位。
> 入力範囲: **差分のみ**（既存実装への指摘はしない）。
> 想定モデル: 上位モデル。
> 旧 review スキルの「外部API連携」観点が ADR 0006 の観点分割時に脱落していたため復活。
> OpenAI互換クライアント実装（Issue #7）の前提として必要（Issue #46）。

| No | チェック項目 | 区分 | 重要度 | 備考 |
|---|---|:---:|:---:|---|
| 1 | 外部API呼び出しにリトライ（指数バックオフ等）が実装されている | 手動 | Should | 推論サーバの一時的な過負荷・再起動を想定 |
| 2 | レート制限（429等）を考慮した待機/リトライ制御がある | 手動 | Should | |
| 3 | 外部APIのエラーレスポンス（4xx/5xx）を握りつぶさず適切にハンドリングしている | 手動 | Must | |
| 4 | `requests`/`httpx` 等の呼び出しに `timeout=` が設定されている | 自動 | Must | ruff S113（[`03-security.md`](03-security.md) と重複するため自動区分はそちらでも検出) |
| 5 | OpenAI互換クライアントとしてモデル/ランタイムを差し替えてもレスポンス形式の差異を吸収できる | 手動 | Should | ADR 0002 |

> 自動区分は pre-commit/CI が green であればスキップ可。
